Semantic Catalogs of LCA Data

Here is a listing of the available catalogs. This page is not automatically generated so it may be out of date. Please view the GitHub repo for the most current information.

These files are GZIP-compressed JSON files.

Unit Process Data

These catalogs contain Unit Process data (exchanges and exchange values) for freely available data sources.
  • US LCI Database: uslci_ecospold.json.gz (Data Source Type: EcoSpold v1)
  • ELCD Database: elcd_3.2.json.gz (Data Source Type: ILCD)

Reference Data

These catalogs contain listings of unit processes, including reference exchanges but excluding other exchanges and exchange values).
  • Ecoinvent 3.2 Undefined: ecoinvent_3.2_undefined_xlsx.json.gz (Data Source Type: Ecoinvent Activity Overview spreadsheet)
  • Ecoinvent 3.2 APOS: ecoinvent_3.2_apos_xlsx.json.gz (Data Source Type: Ecoinvent Activity Overview spreadsheet)
  • Ecoinvent 3.2 Consequential (Long-term): ecoinvent_3.2_consequential_xlsx.json.gz (Data Source Type: Ecoinvent Activity Overview spreadsheet)
  • Ecoinvent 3.2 Cutoff: ecoinvent_3.2_cut-off_xlsx.json.gz (Data Source Type: Ecoinvent Activity Overview spreadsheet)
  • GaBi Professional 2016: gabi_2016_professional-database-2016.json.gz (Data Source Type: ILCD web archive)
  • GaBi Extensions 2016: gabi_2016_all-extensions.json.gz (Data Source Type: ILCD web archive)

LCIA Implementations

These catalogs contain flow characterization data for freely available LCIA implementations, including quantities and characterization factors, supporting locale-specific LCIA factors.
  • Ecoinvent LCIA Implementation (v3.1 2014-08-13): ei_lcia.json.gz (Data Source Type: Ecoinvent LCIA spreadsheet)
  • ELCD LCIA: elcd_lcia.json.gz (Data Source Type: ILCD) (only includes CFs for elementary flows that appear as exchanges in the ELCD database)

Authors and Contributors

The catalogs were produced in connection with a manuscript under review at the Journal of Cleaner Production, authored by Brandon Kuczenski (bkuczenski@ucsb.edu), Christopher B. Davis (TU Groningen), Beatriz Rivela (Inviable.es), and Krzysztof Janowicz (UCSB Department of Geography).
